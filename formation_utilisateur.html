Ajout d'autres utilisateurs

Félicitations ! 😃

Vous venez d’acquérir "Ma Gestion Zen" ! Ce logiciel sera un atout précieux dans la gestion de votre auto-école. Pour exploiter pleinement ses fonctionnalités, il est essentiel de comprendre comment créer et gérer des utilisateurs. Cette leçon vous guidera à travers le processus de création d'un nouvel utilisateur, en vous montrant comment attribuer les niveaux d'accès appropriés pour optimiser la collaboration au sein de votre établissement.

Objectif de la Leçon : 💪

À la fin de cette leçon, vous serez en mesure de créer de nouveaux utilisateurs dans le logiciel "Ma Gestion Zen" pour collaborer efficacement à la gestion de votre auto-école.

Aperçu du Contenu : 🫣

Dans cette leçon, nous couvrirons les éléments suivants :

  • Différenciation des Interfaces : Interne et Moniteur.
  • Niveaux d'accès : Visiteur, Administrateur, Directeur, Enseignant et Agent.
  • Suppression d'utilisateur : Désactivation plutôt que suppression.
  • Création d'un nouvel utilisateur, y compris les détails spécifiques à remplir.

1. Différenciation des Interfaces : 🤯

  • L'interface "Interne" est destinée au personnel administratif et offre des fonctionnalités avancées, y compris la gestion de la base de données, des finances, etc.
  • L'interface "Moniteur" est conçue pour les enseignants et leur permet de gérer les leçons et les horaires.

2. Niveaux d'Accès : 😎

  • Les niveaux d'accès sont adaptés aux rôles spécifiques :

    • Visiteur : Accès limité, généralement pour les parents d'élèves.
    • Administrateur : Accès complet avec tous les droits.
    • Directeur : Accès limité, avec certaines restrictions comme l'impossibilité de préparer des leçons.
    • Enseignant : Accès pour les enseignants, incluant la gestion des leçons et du planning.
    • Agent : Accès pour les tâches administratives.

Tous les niveaux d'accès peuvent être paramétrer dans la section Droits.

3. Création d'un Nouvel Utilisateur : 🤓

  • Pour ajouter un nouvel utilisateur, suivez ces étapes :
    • Allez dans l'onglet "Configuration" dans la barre de menu principal MGZ, puis sélectionnez "Configuration Application."


  • Cliquez sur l'onglet "Général" dans la barre latéral, puis "Utilisateur."
  • Utilisez la barre grise en haut pour ajouter un nouvel utilisateur.


  • Sélectionnez l'interface de l'utilisateur (Interne ou Moniteur).
  • Remplissez les informations nécessaires, telles que le prénom, le nom, l'identifiant (unique), le mot de passe, l'email, le téléphone.
  • Assurez-vous de cocher la case "Actif" pour que l'enseignant apparaisse sur le planning.
  • Cliquez sur "Ajouter" pour créer le nouvel utilisateur.

Vous remarquerez que lorsque vous créer un utilisateur le mot de passe choisi n'ai pas celui qui apparait dans la liste une fois crée 😱 pas de panique, ce n'ai qu'une simple sécurité par encryptage pour ne pas divulger le mot de passe.

Pour le modifier vous n'avez qu'a selection le mot de passe crypté et écrire par dessus, et surtout n'oubliez pas de cliquer sur ENREGISTRER ! 😜



4. Suppression d'Utilisateur : ❌

  • Il n'est pas possible de supprimer un utilisateur, mais vous pouvez le désactiver.
  • Pour désactiver un utilisateur, cochez la case "Actif" dans la colonne correspondante. La case sera verte avec un "Oui" lorsqu'il est actif, et en cliquant dessus, l'utilisateur sera désactivé.


5. Conclusion : 😁

  • Vous avez maintenant acquis les compétences nécessaires pour créer un nouvel utilisateur dans "Ma Gestion Zen" en attribuant les niveaux d'accès appropriés.
  • Il est important de personnaliser les autorisations en fonction des besoins de chaque utilisateur.
  • Gardez à l'esprit que les utilisateurs désactivés peuvent être réactivés à tout moment.

Avec ces instructions détaillées, vous serez en mesure de gérer efficacement les utilisateurs dans "Ma Gestion Zen" pour optimiser la gestion de votre auto-école.
